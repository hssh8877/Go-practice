# Contact Manager in Go

A simple command-line contact manager written in Go.  
You can add, view, search, delete, save, and load contacts using JSON files.

## Features

- Add new contacts (Name, Email, Phone)  
- List all contacts  
- Search contacts by name  
- Delete a contact  
- Save contacts to a JSON file (`contacts.json`)  
- Load contacts from a JSON file  

# Installation

Clone  the repository
```bash
git clone https://github.com/hssh8877/Go-practice
cd Go-practice/contacmanager
```

# Usage
```bash
go run .
```
