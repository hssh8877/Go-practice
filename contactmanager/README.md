# Contact Manager in Go

A simple command-line contact manager written in Go.  
You can add, view, search, delete, save, and load contacts using JSON files.

## Features

- Add new contacts (Name, Email, Phone)  
- List all contacts  
- Search contacts by name  
- Delete a contact  
- Save contacts to a JSON file (`contacts.json`)  
- Load contacts from a JSON file  

## Usage

1. Run the program:  
   ```bash
   go run main.go contacts_save_load.go

2. The Menu:
   ===Contacts===
   1. Add contact
   2. Show contacts
   3. Search contact
   4. Delete contact
   5. Save contacts
   6. Load contacts
   7. Exit


3. Enter the number of your choice and follow the prompts.
